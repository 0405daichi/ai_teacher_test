簡単に経歴を申し上げます。
-高校卒業後プログラミングを学習しソフトウェア会社で働くために福岡から名古屋へ
-約1年後、ある目的のため東京へ
-アルバイトで生活しながら事業を始めようと数ヶ月作業に取り組むが断念
-別のやり方で目的を果たすため、個人でアプリを開発

僕の目的は、『学生時代のあの日の僕のような人を救うこと』です。
そのために僕は、ある現象を起こすもしくはそれを促進したいと思っております。
その現象とは、『学校を一つの手段（例えば習い事や学童）と捉える人が増える』というものです。
僕は、「黒板が電子黒板になった」「その場に居なくても授業が受けられるようになった」「技術革新によって既存の学校教育を最適化すべきだ」といった表面的なことを世の中に伝えたいのでは決してありません。
"当時既にぐうの音も出ないほど加速していた世界"を具現化したもの、それによる気づきの可能性を、人の人生で目のつくところに散りばめたいと思っております。例えば学校の中の狭い世界で苦しんでいる人、その人を反抗期で片付けたり陳腐化した価値観で苦しめたりする人にです。
そしてその先に、学校・教育に対する認識の変化があれば本望です。

孫さんは、後世への遺物、人が動き出すきっかけを「冒険の書」などの様々な形で遺されているかと思います。
その遺物の裏にある数えきれないほどの思考の数や重さが、そこに確かに在ると恐縮ながら共感いたしました。
やり方は違えど、同じ姿勢であると感じております。
ぜひ一度、お話しする機会をいただけないでしょうか。
